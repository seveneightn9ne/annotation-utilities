Instructions

Search for sequences of words, universal/PTB part-of-speech tags and English UD relation labels
Regular expressions are supported for searching words.

Examples

  • see it matches the string "see it"
  • see DET NOUN matches "see that show", "see the sign", etc.
  • \w+ing something matches "seeing something", "seeking something", etc.
  • amod NNS matches adjectival modifier followed by a plural noun, such as "best cakes", "bigger halls", etc.

Filters

Filter query results to sentences that contain a specific gramattical error and/or specific native language. Using filters when the query is empty will retrieve all the sentences that correspond to the filter.
